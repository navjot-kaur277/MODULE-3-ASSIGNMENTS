# React FirstComponent Assignment

## Overview
This project demonstrates a basic React setup using Vite. It renders a custom component called `FirstComponent`.

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Visit `http://localhost:5173`
